# 58同城租房数据分析的前端
## 说明
  由后端提供接口，完全的前后端分离,后端使用Springboot+Mybatis,数据由爬虫爬取，这里只有前端代码
## 开发环境
前端开发框架：Vue+eCharts
集成开发环境：Visual Studio Code
## 运行前准备工作
1.cmd：找到该文件  
  npm run serve  
或者VSCode打开该文件  
2.浏览器输入http://localhost:8889/
